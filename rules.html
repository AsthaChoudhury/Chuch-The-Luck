Chuck-a-luck is a luck analysis game which calculates the luck factor based on the total number of correct guesses made by the player

  • Rules of the game:
  • The number from 0-99 will be displayed on the player on the screen
  • Below the number, 3 options will be given in which only one option contains a correct number which is displayed above and 2 options will have wrong numbers.
  • The player will have to click on any option among those 3 to check if displayed number and the number hidden in the option gets matched. Once the user click on any option the hidden numbers below all the options will be shown on the screen
  • If the displayed number and the number shown after the user click on the option gets matched, it will increase the count of Lucky attempt. However if the number couldn't get matched, it will increase the count of wrong attempts
  • Total 10 attempts will be given to the user to check the luck factor. Based on the lucky attempts or wrong attempts the luck factor will be calculated
  • Finally once the user give the attempts the calculated result will be displayed on the screen
